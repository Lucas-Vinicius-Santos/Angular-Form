# Getting Started

```bash
# Clone Repository
$ git clone https://github.com/Lucas-Vinicius-Santos/Angular-Form.git

# Install Dependencies
$ npm install

# Run Aplication
$ ng start
```
:eyes: Go to http://localhost:4200/ to see the result.
